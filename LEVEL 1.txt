Ello this is some notes for myself regarding the first level of my projects (Code + Docker + Kubernetes)

First, Create the project file:

cd ~/Documents

mkdir todo-app
cd todo-app

# create a Python virtual environment
python -m venv venv

# activate it (I'll do this every time I work on the project)
venv\Scripts\activate

pip install flask

# save the dependencies to a file
pip freeze > requirements.txt

# set up Git
git init
echo venv/ > .gitignore
git add .
git commit -m "initial commit"
----------------------------------------------------------------------------------------------------------
Then Testing on a second terminal tab:
on terminal:

# 1. Health check
Invoke-RestMethod -Uri http://localhost:5000/health

# 2. Add a to-do
Invoke-RestMethod -Uri http://localhost:5000/todos -Method POST -ContentType "application/json" -Body '{"task": "Learn DevOps"}'

# 3. Get all to-dos
Invoke-RestMethod -Uri http://localhost:5000/todos
----------------------------------------------------------------------------------------------------------
After Creating the Dockerfile, build the docker image:
docker build -t todo-app:v1 .
----------------------------------------------------------------------------------------------------------
Run the container:
docker run -p 5000:5000 todo-app:v1
----------------------------------------------------------------------------------------------------------
Then test again, also write this to show the running container:
docker ps
----------------------------------------------------------------------------------------------------------
Then send it to docker hub
docker login
docker tag todo-app:v1 YOURUSERNAME/todo-app:v1
docker push YOURUSERNAME/todo-app:v1
----------------------------------------------------------------------------------------------------------
Start Minikube → Create Deployment → Create Service → Access App → Test it
minikube start --driver=docker
then verify:
minikube status
----------------------------------------------------------------------------------------------------------
After finishing deployment.yaml and service.yaml do this on terminal:
# Apply the deployment
kubectl apply -f C:\Projects\todo-app\k8s\deployment.yaml

# Apply the service
kubectl apply -f C:\Projects\todo-app\k8s\service.yaml
----------------------------------------------------------------------------------------------------------
now test:
kubectl get pods
kubectl get services
kubectl get pods --watch
----------------------------------------------------------------------------------------------------------
Now access the app
minikube service todo-app-service --url
